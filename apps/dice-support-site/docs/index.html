Chat Bubble

Ready
How do I send an invoice?
To send your first invoice, go to Business > Invoices and click New Invoice. You can also save as draft if you're not ready to send yet.
Purpose

User and system message bubbles for conversational UI. User messages are blue with white text, aligned right. System messages are plain text, aligned left.

Fullscreen Mode

In fullscreen mode, the content column is 800px max centered. User messages are constrained to 480px max-width, while system messages expand to fill the column.

HTML
<div class="message message--user">
  <div class="chat-bubble chat-bubble--user">
    User message here
  </div>
</div>

<div class="message message--system">
  <div class="chat-bubble chat-bubble--system">
    System response here
  </div>
</div>
CSS
components/chat/ChatBubble.css
