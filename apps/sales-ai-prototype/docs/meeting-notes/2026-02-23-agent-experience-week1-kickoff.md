# Agent Experience System: Week 1 Kickoff

**Date:** 2026-02-23, 3:30 PM
**Context:** Six-week sprint designing agent experiences across Xero

---

## Key Principles

- **Design for target state**, not MVP -- leadership will descope later
- Agent platform proof of concept running in parallel (6 weeks)
- No build work during this period; build follows immediately after

## Ownership Model

**Pattern Owners** -- Design reusable interaction patterns
**Agent Owners** -- Build end-to-end experiences using patterns

### Weekly Cadence
- **Tuesdays:** Pattern design review (pattern owners present, get feedback)
- **Thursdays:** Agent design review (agent owners present, inform pattern refinement)
- **Slack:** Agent Experience Working Group (private), public channel for stakeholder updates
