## Pulling channel content into the repo (use Glean)

**Glean is the main way to pull channel content.** This repo can't call Glean or Slack directly; you run Glean, then paste the results.

1. **Glean (recommended):** In Glean, search Slack for this channel (e.g. channel ID `C0AGMBC51U0`, or "bill ai" / "bill pay agent" in that channel). Copy the conversation(s) or export → **paste into** [docs/meeting-notes/2026-02-24-slack-channel-C0AGMBC51U0-from-glean.md](../meeting-notes/2026-02-24-slack-channel-C0AGMBC51U0-from-glean.md). That doc is a ready-to-fill template; once pasted, we structure it (threads, dates, action items) and link it below.
2. **Export (workspace admins):** Slack workspace export (Settings → Import/Export) → copy threads or history into the same meeting note or a new `yyyy-mm-dd-slack-channel-C0AGMBC51U0.md`.
3. **Copy-paste from Slack:** Select messages in the channel → copy → paste into the same meeting note or a new one.

After you have raw content in a meeting note, we add the doc to INDEX and link it under "Captured channel snapshots" below.
