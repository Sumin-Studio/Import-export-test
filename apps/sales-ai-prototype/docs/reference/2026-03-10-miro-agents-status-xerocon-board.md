# Miro: Agents status / Xerocon board (10 Mar 2026)

**Overview:** How to use (or create) a Miro board for agents status and Xerocon story. Use Miro to aggregate Confluence snippets, Figma links, and video thumbnails; run show-and-tell or voting in sync; don’t duplicate long-form content — link to Confluence.

## When to use a Miro board

- **Aggregate** Confluence snippets, Figma links, and video thumbnails in one visual place.
- **Show-and-tell or voting** in the Platform & Eco Design Sync (or a dedicated Agents sync): one board or frame per theme, 5–10 min tour.
- **Brainstorm** with the team around “How might we…” or “This week’s focus.”

## Suggested structure (Agents status / Xerocon)

- **One board** (or one frame per sync) for “Agents status” or “Xerocon story.”
- **Short framing** at the top: e.g. “This week’s focus” or “How might we tell the Xerocon story?”
- **Cards or screenshots** that **link** to Confluence pages, Figma, repo (no full long-form copy on the board).
- **Optional:** Voting (hearts/stars) and a 5–10 min “tour” in the sync to discuss.

## Where to link

- From **Confluence:** Add a “Key links” row or section on the Payments Agents Hub (or the ritual page for “Agents design sync”) with the Miro board URL.
- From **Slack:** When you share a material update, you can link both Confluence and Miro in the same post if the board is the primary artifact for that update.

## Creating the board

If you create a new board:

1. Create it in the same Miro space/team as the existing Platform & Eco Design Sync board (or duplicate that board’s structure).
2. Add the board URL to this doc and to `docs/status/INDEX-whats-stored-and-next-steps.md` under “What’s stored.”
3. Add the link to the Confluence hub “Key links” or the Agents design sync ritual page so the team can find it.
