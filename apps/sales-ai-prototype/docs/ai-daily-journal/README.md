# AI Daily Journal

**Purpose:** One file per day (`yyyy-mm-dd.md`) with a **detailed** summary of what was discussed that day — meetings, Slack, decisions, and key documents. Kept so there is a single place to see "what happened on this date."

**Convention:**

- Add or update the file for **today** whenever something material is discussed (meeting, thread, decision).
- Each entry should include: **What was discussed** (who, what, outcomes), **Decisions**, and **Key documents** (links to meeting-notes, workshop, strategy, etc.).
- Weekend or quiet days can be a short "No formal meetings captured" plus any artifacts dated to that day.

**Files:** 2026-02-10.md through 2026-02-17.md (and onward). Start with the most recent date for "what did we just do?"
