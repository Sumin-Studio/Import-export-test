# Static Files Local Preview

Instructions for helping the user preview an app that contains only static files (HTML/CSS/JS) without a build system.

## When to Use

When the user is working on an app that has only static files and needs to preview it locally.

## Steps

1. Suggest using a local web server.
2. If on macOS, suggest opening the Terminal and running `python3 -m http.server 8000` in the app folder.
3. If on Windows, suggest using a server extension for VS Code, like `ritwickdey.LiveServer`.
